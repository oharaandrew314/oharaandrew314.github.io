About Me

Hi, my name is Andrew O'Hara. I'm a pending BEng Software Engineering graduate from Carleton University, starting my first full-time position at ProntoForms as a Cloud Application Server Developer.

Java and Python are my favourite languages to work with. All of my personal projects so far have been General-User desktop applications, and thus, Java is the best choice due to platform agnosticism and minimal dependency hassle. However, when I need a quick script to do something right now (especially when functional programming will be useful), then my go-to is definitely Python.

I love working on my own open source projects, and am very proud of my latest release, which is a Mod Manager for Kerbal Space Program, written in Java. It currently has over 4 000 downloads, and I am eager to gain an even larger following as I continue its development.

Outside of my developer life, I love to play Role-Playing Games like Mass Effect, cook, tweak my PC, and take care of my cat, Tigger. In the Winter, I'm an avid skater and skiboarder, and in all seasons, casual yogi.

Other Projects

360verify

Cloud-Based Data Validation Tool for 360pi inc. Revolutionized the existing manual data validation workflow by automating most of the process. Jobs are automatically added to the server via REST API, processed in the background, and then made available for trained workers to log in and perform the portion of testing that requires a human's judgement. It is currently used in production by trained outsourced workers.
