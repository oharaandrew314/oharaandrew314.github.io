Other Projects

TinkerTech - Factorio Mod Manager Open Source

Java Mod Manager for Factorio. Unfortunately, the developers announced an integrated manager soon after I began development. However, I managed to learn a lot about Java's new GUI framework, Java FX. Most interestingly, I learned how to combine FX's FXML controllers with Guice Dependency injection. I wish I could have investigated this symbiotic relationship further, but there was no more need for this app, so it died in its infancy.

Find it on Github

TinkerTime - KSP Mod Manager Open Source

Java Mod Manager for Kerbal Space Program. It was perhaps one of my proudest achievements, which accumulated over 4 000 downloads during its lifetime. However, I eventually stopped playing the game, and decided to end support.

Find it on Github

The Wheel of Time - Conquest Open Source

Fan game based off of the Wheel of Time Novel Series, in the fashion of a Fire Emblem Game. Developed on top of an in-house 2D grid game engine. It currently looks and sounds beautiful, and is somewhat playable, but development was put on hold to pursue TinkerTime.

Find it on Github
